How do I install these mods? +

Step-by-Step Installation:
1. Click the download button on any mod
2. The .jar file will be saved to your computer
3. Make sure you have Fabric or Forge installed for Minecraft 1.21
4. Place the .jar file in your Minecraft 'mods' folder
5. Launch Minecraft with the mod loader
6. Join Donut SMP and enjoy!
